## Book Personalized Recommendation System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/3abbf3c87f24429e9f9da63a575fe167/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Book Information
5. Good Book Recommendations
6. Message Feedback
7. Personal Center
8. Background Management
9. Book Reservation Management
10. Book Return Management
11. My Collection Management


**Administrator Functions:**
1. Login
2. Personal Center
3. Student Management
4. Book Category Management
5. Book Information Management
6. Book Reservation Management
7. Book Return Management
8. Administrator Management
9. Message Board Management
10. System Management
